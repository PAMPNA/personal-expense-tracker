# Personal Expense Tracker

A cross-platform Personal Expense Tracker application where users can add daily expenses using a Flutter mobile app and analyze spending through an Angular web dashboard powered by a Node.js backend.

## Tech Stack
- Flutter (Mobile Application)
- Angular (Web Dashboard)
- Node.js with TypeScript (Backend APIs)

## Features
- Add, edit, and delete expenses
- Category-wise expense tracking
- Monthly spending analytics
- Responsive UI for mobile and web
- Centralized backend API

## Project Structure
- backend/ → Node.js + TypeScript APIs
- expense-tracker-web/ → Angular dashboard
- expense_tracker_flutter.zip → Flutter mobile app
